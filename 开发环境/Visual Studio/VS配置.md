## 正则搜索
`.` : 匹配任意字符  
`*` : 匹配一个字符任意次  
`.*` : 匹配任意字符串  
开启正则搜索后(搜索框下面一个icon为`.*`的按钮),可如下搜索:  
```
Activate.*By
TryActivateAbilitiesByTag
```

## 死循环
程序死循环时,可以通过VS暂停程序,看程序停在那里,来判断死循环位置  
暂停按钮就在停止按钮的左边
